# Skincare Clinic Promotion Agent System

## 에이전트 역할 정의서

이 시스템은 소규모 피부관리실의 온라인 홍보 콘텐츠를 자동 생성하는 멀티 에이전트 시스템이다.

---

## 에이전트 구성

### 1. Orchestrator (오케스트레이터)
- **파일**: `src/agents/orchestrator.py`
- **모델**: Claude API
- **역할**: 사용자 요청을 분석하고 플랫폼별 에이전트에 작업을 분배하는 메인 코디네이터
- **책임**:
  - 콘텐츠 주제 분석 및 키워드 추출
  - 플랫폼별 에이전트 병렬 호출
  - 결과 통합 및 검수용 미리보기 제공
  - 샵 프로필 정보를 각 에이전트에 전달

### 2. Blog Agent (블로그 에이전트)
- **파일**: `src/agents/blog_agent.py`
- **모델**: Claude API / OpenAI GPT (용도별 분담)
- **역할**: 네이버 블로그 SEO 최적화 포스트 생성
- **출력**: 제목, 본문(마크다운), 태그, 키워드
- **콘텐츠 유형**:
  - 피부 고민별 솔루션 포스트 (여드름, 모공, 주름, 색소 등)
  - 시술 전/후 관리 효과 포스트
  - 피부 타입별 홈케어 가이드
  - 이벤트/프로모션 포스트

### 3. Instagram Agent (인스타그램 에이전트)
- **파일**: `src/agents/instagram_agent.py`
- **모델**: OpenAI GPT (감성적 카피), Gemini 2.5 (이미지)
- **역할**: 인스타그램 피드/릴스용 콘텐츠 생성
- **출력**: 캡션, 해시태그(최대 30개), 이미지 설명, 릴스 스크립트
- **콘텐츠 유형**:
  - 피드 포스트 (관리 과정, 결과물)
  - 릴스 스크립트 (스킨케어 팁, 관리 하이라이트)
  - 캐러셀 (피부 타입별 가이드)

### 4. YouTube Agent (유튜브 에이전트)
- **파일**: `src/agents/youtube_agent.py`
- **모델**: Claude API (스크립트), Kling AI (영상)
- **역할**: 유튜브 쇼츠/영상 콘텐츠 생성
- **출력**: 영상 스크립트, 제목, 설명, 태그, 썸네일 텍스트
- **콘텐츠 유형**:
  - 쇼츠 (60초 이내) — 스킨케어 팁, 관리 하이라이트
  - 일반 영상 (5-10분) — 관리 전과정, 피부 타입별 가이드
  - ASMR 관리 영상

## AI 모델 역할 분담

| 용도 | 모델 | 이유 |
|------|------|------|
| 콘텐츠 기획/전략 | Claude API | 구조적 분석, 전략 수립에 강점 |
| 블로그 본문 | Claude API / GPT | 긴 글 작성, SEO 최적화 |
| 인스타 캡션 | GPT | 감성적 짧은 카피에 강점 |
| 유튜브 스크립트 | Claude API | 구조화된 스크립트 작성 |
| 이미지 생성 | Gemini 2.5 | 관리 이미지, 프로모션 배너 |
| 영상 생성 | Kling AI | 유튜브 쇼츠, 인스타 릴스 |

---

## 워크플로우

```
[사용자 입력] → [Orchestrator] → [플랫폼별 에이전트 병렬 실행]
                                        ↓
                                 [텍스트 + 이미지 + 영상 생성]
                                        ↓
                                 [콘텐츠 통합/미리보기]
                                        ↓
                                 [사용자 검수/수정]
                                        ↓
                                 [output/ 저장 → 수동 업로드]
```

## 설정

- 샵 프로필: `config/shop_profile.yaml`
- API 키: `.env` 파일
- 콘텐츠 템플릿: `templates/` 디렉토리

## 주의사항

- **의료 행위 암시 금지**: "진단", "치료", "처방" 등의 의료 용어 사용 금지
- **과장 표현 금지**: 비현실적인 효과 주장 금지
- **비포애프터**: 실제 관리 효과를 솔직하게 전달, 과장 금지
