# Core Beliefs — 콘텐츠 철학

## 피부관리실 홍보 콘텐츠의 핵심 원칙

### 1. 진정성
- 과장하지 않는다
- 실제 관리 결과를 솔직하게 전달한다
- 의료 행위를 암시하지 않는다 ("치료", "진단" 등 금지)
- 비포애프터는 현실적인 범위 내에서만

### 2. 전문성
- 피부 타입과 고민에 대한 정확한 정보를 제공한다
- 피부과학 기반의 설명 (세포 재생, 콜라겐, 수분 장벽 등)
- 전문 용어를 쉽게 풀어 설명한다
- 관리사의 경력과 전문성을 자연스럽게 어필

### 3. 공감
- 피부 고민에 대한 공감으로 시작
- "저도 그런 경험이 있어요" 식의 따뜻한 접근
- 고객의 불안감을 해소하는 정보 제공
- 결과뿐 아니라 과정도 중요하게 전달

### 4. 실용성
- 가격, 위치, 영업시간 등 실용 정보 필수 포함
- 홈케어 팁을 함께 제공 (관리 전후 주의사항)
- 예약/상담 방법을 명확히 안내
- CTA(Call to Action)는 자연스럽게
