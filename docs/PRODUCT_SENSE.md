# Product Sense

## 타겟 사용자
- **소규모 피부관리실 운영자** (1-3인 운영)
- 마케팅 전담 인력이 없는 관리실
- SNS 콘텐츠 제작에 시간을 많이 투자하기 어려운 상황
- 기본적인 PC/스마트폰 활용이 가능한 수준

## 핵심 가치
1. **시간 절약**: 3개 플랫폼 콘텐츠를 한 번에 생성
2. **전문성**: SEO 최적화, 해시태그 전략, 피부관리 전문 용어 내장
3. **일관성**: 브랜드 톤앤매너를 유지하는 콘텐츠
4. **접근성**: CLI 한 줄로 콘텐츠 생성
5. **안전성**: 의료법 위반 표현 자동 차단

## 콘텐츠 전략

### 블로그 (네이버)
- 목적: 검색 유입, 신규 고객 확보
- 전략: C-Rank/D.I.A 알고리즘 대응 SEO
- 핵심: 피부 고민별 솔루션 + 관리 과정 상세 설명
- 빈도: 주 2-3회

### 인스타그램
- 목적: 브랜드 인지도, 커뮤니티 형성
- 전략: 비주얼 중심 + 공감 카피 + 해시태그 최적화
- 핵심: 관리 전후 변화, 클린한 분위기
- 빈도: 주 3-5회 (피드+릴스+스토리)

### 유튜브
- 목적: 관리 과정 시연, 전문성 신뢰 구축
- 전략: 쇼츠로 진입, 일반 영상으로 전환
- 핵심: ASMR 관리 영상, 피부 타입별 가이드
- 빈도: 주 1-2회
